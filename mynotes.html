Set up and installation

  1. Install Git
  2. Find your terminal program
  3. Install Virtual Box
  4. Install vagrant
  5. Download VM configuration file. The configuration file, called Vagrantfile, is here: Download!
  6. go to folder in terminal example: "cd Desktop/shell". type vagrant up and then vagrant ssh.
